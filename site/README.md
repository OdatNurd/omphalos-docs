This repository represents what used to be an automatically deployed
documentation set for the [Omphalos](https://github.com/odatnurd/omphalos)
project and which is now a more manually deployed version, and  contains what
was an automatically generated static version of the  documentation site.

The site itself is live at https://omphalos.ruinouspileofcrap.com.

If you would like to help with documentation (corrections, amendments, new
sections, etc) please see the above mentioned repository.
